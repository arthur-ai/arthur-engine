# Customer Support Agent - Automation Scripts

## Quick Reference

### Bootstrap (One Time)
```bash
cd .. && yarn bootstrap:extract
```

### Create New Task
```bash
cd .. && yarn setup:new-task "My Task Name"
```

### Batch Create
```bash
./example-create-demo-tasks.sh
```

---

## Scripts in This Directory

| Script | Purpose | Usage |
|--------|---------|-------|
| `bootstrap-extract.ts` | Extract config from existing task | Called by runner |
| `run-bootstrap-extract.js` | Bootstrap runner with env loading | `node run-bootstrap-extract.js` |
| `setup-new-task.ts` | Create new complete task | Called by runner |
| `run-setup-new-task.js` | Setup runner with env loading | `node run-setup-new-task.js "Task Name"` |
| `example-create-demo-tasks.sh` | Batch creation example | `./example-create-demo-tasks.sh` |

---

## Data Directories

### `bootstrap-data/`
Generated by bootstrap extract. Contains:
- `task.json` - Task configuration
- `prompts.json` - 5 complete prompt templates
- `eval-definitions.json` - 3 eval definitions
- `test-questions.json` - 10 test questions

### Generated Files
- `setup-result.json` - Latest task creation results

## Quick Start

```bash
# 1. Bootstrap (one time)
cd ..
yarn bootstrap:extract

# 2. Create task
yarn setup:new-task "My Demo"

# 3. Update .env
# ARTHUR_TASK_ID=<new-task-id>

# 4. Test
yarn test:questions
```
